# 🧭 인생 2막 나침반 (은퇴 준비 지수 진단 RRI V2)

**"더 찬란하게 빛날 당신의 인생 2막, 나침반이 함께합니다."**

'인생 2막 나침반'은 은퇴를 준비하거나 이미 시작하신 분들이 자신의 준비 상태를 객관적으로 점검하고, 맞춤형 전략을 세울 수 있도록 돕는 디지털 진단 플랫폼입니다. 

---

## 👋 서비스 소개

단순한 자산 비교를 넘어, **재무·건강·관계·활동·환경** 5가지 핵심 영역을 입체적으로 분석합니다. V2 업데이트를 통해 더 정교해진 알고리즘과 아름다운 디자인으로 돌아왔습니다.

- **8가지 인생 페르소나**: 설문 결과에 따라 당신의 은퇴 스타일을 '즐거운 여행자', '조용한 수호자' 등 8가지 캐릭터로 매칭해 드립니다.
- **프리미엄 데이터 시각화**: 고도화된 Radar Chart와 영역별 분석 바를 통해 나의 강점과 보완점을 직관적으로 파악하세요.
- **스마트 테마 & 모바일 최적화**: 다크 모드와 라이트 모드를 완벽 지원하며, 어떤 모바일 기기에서도 앱처럼 부드럽게 작동합니다.
- **결과 소장 및 공유**: 진단 결과를 고화질 이미지로 저장하거나, 친구 및 가족에게 링크로 바로 공유할 수 있습니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
| :--- | :--- |
| **8대 페르소나 진단** | 상황별 시나리오 질문을 통해 나의 은퇴 성향과 캐릭터를 도출 |
| **통합 레이더 분석** | 5대 영역별 준비도 점수를 시각적으로 비교 분석 |
| **맞춤형 보완 솔루션** | 각 영역별 점수에 따른 구체적이고 실행 가능한 '처방전' 제공 |
| **이미지 저장 & 공유** | 터치 한 번으로 리포트를 이미지로 저장하거나 링크 복사 가능 |
| **지능형 사용자 경험** | 테마 기억 기능(LocalStorage) 및 60fps 부드러운 애니메이션 |

## 🚀 시작하기

1. **[진단 페이지 접속]**: [https://happyretire.github.io/Retirement-Readiness-Index-V2/](https://happyretire.github.io/Retirement-Readiness-Index-V2/) 클릭
2. **[설문 진행]**: 10가지 상황별 질문에 당신의 생각을 선택해 주세요. (약 3분 소요)
3. **[결과 확인]**: 당신의 은퇴 캐릭터와 종합 분석 리포트를 확인합니다.
4. **[리포트 활용]**: 이미지를 저장하여 소장하거나, 공유 기능을 통해 가족과 함께 계획을 세워보세요.
